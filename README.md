# Jayden's Sudoku (제이든의 스도쿠)

Oju(6세)를 위한 맞춤형 스도쿠 게임 프로젝트입니다.

## 🌟 주요 기능 (Features)

*   **다양한 난이도**: 4x4, 6x6, 9x9 등 연령별 맞춤 크기 제공
*   **어린이 친화적 UI**: 밝은 색상, 큰 버튼, 직관적인 아이콘 사용
*   **스마트 입력 시스템**:
    *   터치 친화적인 팝업 키패드
    *   메모 기능 (후보 숫자 적기)
    *   힌트 모드 (틀린 숫자 즉시 피드백)
*   **인쇄 기능 (Print Mode)**: A4 용지에 최적화된 학습지 인쇄 기능
    *   4x4: 12문제
    *   6x6: 12문제 (최적화 레이아웃)
    *   9x9: 6문제
*   **사운드 효과**: 즐거운 입력음과 승리 효과음 (Web Audio API 사용)

## 🛠 기술 스택 (Tech Stack)

*   **HTML5 / CSS3**: 반응형 디자인 및 인쇄 스타일링 (@media print)
*   **JavaScript (Vanilla)**: 게임 로직, 스도쿠 생성 알고리즘, Web Audio API 사운드 구현

## 🚀 실행 방법 (How to Run)

1.  `index.html` 파일을 웹 브라우저(Chrome, Edge 등)에서 엽니다.
2.  게임을 즐기거나 '인쇄' 버튼을 눌러 학습지로 활용합니다.
